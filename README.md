# 🌙 Demir's Night Vision

## ✨ Features
 
| | |
|---|---|
| 🌙 **Toggle** | Press `N` to enable/disable Night Vision instantly |
| ⚙️ **Configurable** | Remap via *Options → Controls → Key Binds* |
| 💬 **Notifications** | <span style="color:green">Night Vision: ON</span> / <span style="color:red">Night Vision: OFF</span> displayed on screen |
| 🎮 **Client-side** | No server installation required |

## 📦 Installation
 
1. Install **Fabric Loader** for your Minecraft version from [fabricmc.net](https://fabricmc.net)
2. Download **Fabric API** and place it in your `mods` folder
3. Download the mod **JAR** and place it in your `mods` folder
4. Launch Minecraft with the **Fabric profile**
> **Mods folder location:**
> - Windows: `%appdata%\.minecraft\mods`
> - Linux: `~/.minecraft/mods`
> - macOS: `~/Library/Application Support/minecraft/mods`
 
---
 
## 🎮 Usage
 
1. Load into any world or server
2. Press `N` → **Night Vision: ON** ✅
3. Press `N` again → **Night Vision: OFF** ❌
4. To remap: *Options → Controls → Key Binds → Night Vision Toggle*
---
 
## 🏗️ Building from Source
 
```bash
git clone https://github.com/demirgitbuh/demirs-night-vision
cd demirs-night-vision
./gradlew build
```
 
Output JAR will be in `build/libs/`.
