Demir's Night Vision Mod

🌙 Toggle Night Vision with the N key ⚙️ Configurable keybind in Options > Controls > Key Binds 💬 On-screen notifications when toggling 🎮 Client-side only - no server required

ABOUT

Minecraft 1.21.1 Fabric Loader 0.16.9+ Fabric API

Installation

Install Fabric Loader for Minecraft 1.21.1 Download Fabric API and place it in your mods folder Download the mod JAR file and place it in your mods folder Launch Minecraft with the Fabric profile

Usage

Enter a world Press N to enable Night Vision (you'll see "Night Vision: ON" in green) Press N again to disable (you'll see "Night Vision: OFF" in red)
